Conversor de Áudio para Texto

Este projeto permite converter arquivos de áudio no formato .ogg (como os do WhatsApp) em texto usando a biblioteca whisper da OpenAI.

Pré-requisitos

Antes de executar o script, certifique-se de ter instalado os seguintes pacotes:

pip install openai-whisper

Como Usar

Execute o script principal:

python audio_to_text.py

Escolha a opção desejada no menu:

Opção 1: Digite o caminho do arquivo .ogg que deseja converter.

Opção 2: Sair do programa.

Selecione um arquivo de áudio e aguarde a conversão.

O texto extraído será exibido na tela.

Exemplo de Execução

Menu de Opções:
1 - Converter áudio para texto
2 - Sair
Escolha uma opção: 1
Digite o caminho do arquivo .ogg: meu_audio.ogg
Convertendo áudio para texto...
Texto extraído:
"Olá, tudo bem? Este é um teste de conversão de áudio para texto."

Problemas Comuns

Arquivo não encontrado: Verifique se o caminho do arquivo está correto.

Erro na conversão: Certifique-se de que o arquivo é um áudio válido e tente novamente.
